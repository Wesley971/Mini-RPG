# 🧙‍♂️ Mini RPG - Projet JavaScript

Bienvenue dans mon **mini jeu RPG textuel**, réalisé en pur **HTML / CSS / JavaScript** !  
Ce projet a été conçu comme un petit défi personnel pour apprendre à gérer des interactions de jeu, des conditions logiques et la progression dans un combat.

## 🎮 Fonctionnalités

- Combat tour par tour : le joueur affronte une série de monstres
- Système de vie et dégâts aléatoires
- Bouton pour **se soigner** pendant le combat
- Plusieurs ennemis à vaincre : gobelin, squelette, ogre... et même un dragon !
- Interface simple et jouable dans un navigateur

## 🛠️ Technologies utilisées

- HTML (structure du jeu)
- CSS (style simple et lisible)
- JavaScript (logique de combat, interactions, aléatoire)

## 🚀 Lancer le jeu

1. Clone ou télécharge ce dépôt
2. Ouvre le fichier `index.html` dans ton navigateur
3. Clique sur "Combattre" ou "Se soigner" et amuse-toi !

```bash
git clone https://github.com/ton-pseudo/mini-rpg.git
```
